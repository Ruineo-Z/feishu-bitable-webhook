# Feishu Bitable Webhook

飞书多维表格 Webhook 后端服务

## Tech Stack

- Bun
- TypeScript
- Hono
- @hono/zod-openapi (自动生成接口文档)

## Getting Started

```bash
# 安装依赖
bun install

# 开发模式
bun dev

# 生产模式
bun start
```

## API Documentation

启动服务后访问：
- Swagger UI: http://localhost:3000/docs
- ReDoc: http://localhost:3000/redoc
